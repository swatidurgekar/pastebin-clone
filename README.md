# Pastebin Clone

A simple Pastebin-like application where users can create text pastes and share a link to view them. Pastes can optionally expire by time or view count.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Upstash Redis (persistence)
- Deployed on Vercel

## Features
- Create a text paste
- Shareable URL
- View paste via API or HTML page
- Optional TTL (time-based expiry)
- Optional max view limit
- Deterministic time support for testing

## Running Locally

1. Install dependencies:
npm install
